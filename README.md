# GruvyEQ

![GruvyEQ Layout Example](Docs/example_image.png)

Minimal EverQuest UI files based on [AYAUI](https://www.eqinterface.com/downloads/fileinfo.php?id=5518) and [Gruvbox](https://github.com/morhetz/gruvbox).

## Install

1. Copy this `gruvyeq` folder into your EverQuest `./uifiles` directory.
2. Copy `UI_Gruvyeq_Example.ini` into the main EverQuest game folder.
3. Start EverQuest and open chat.
4. Run:

```text
/loadskin gruvyeq
```

5. Run:

```text
/copylayout
```

4. Reposition any windows you want and save your layout.

## Notes

- Designed for EverQuest F2P client.
- Includes minimal Player, Target, and Group window variants.
